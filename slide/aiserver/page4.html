시스템 아키텍처 및 기술 스택

ARCHITECTURE_OVERVIEW
계층형 아키텍처 구조 (Layered Architecture) v1.0.0
기술 스택 (Tech Stack)
Frontend
HTML5/CSS3 WebSocket Client Tailwind CSS
Backend
Spring Boot 3.x Spring WebSocket Spring Data JPA
Services
JSch / SSHJ Anthropic API Validation
Data/Infra
H2 (Dev) PostgreSQL (Prod) Docker
데이터 흐름 (Data Flow)

1. 사용자 자연어 입력

Web UI에서 텍스트 또는 음성으로 명령 요청

2. AI 컨텍스트 분석 및 명령 생성

OS/환경 정보를 포함하여 Anthropic API 호출 → 명령어 생성

3. 보안 검증 및 필터링

rm -rf 등 위험 패턴 차단 및 실행 승인 대기

4. SSH 실행 및 스트리밍

JSch를 통해 서버 접속, 실행 로그 WebSocket 실시간 전송

5. 결과 저장 및 피드백

DB에 로그 저장, AI가 결과 요약하여 사용자에게 전달
