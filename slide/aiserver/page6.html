실행 방법

Step-by-Step Installation & Execution Guide

Local Environment

Development Mode

1

사전 준비 사항

REQUIRED

개발 환경 구성을 위한 필수 도구를 설치합니다.

Java Development Kit
JDK 17+
Build Tool
Maven 3.8+
2

설정 파일 수정

CONFIG

src/main/resources/application.properties

Anthropic API 키를 입력하여 AI 기능을 활성화합니다.

anthropic.api.key=sk-ant-api03-xxxxx
보안을 위해 환경변수(ENV) 사용을 권장합니다.
3

프로젝트 실행

BUILD & RUN

Maven Wrapper 또는 명령어를 통해 애플리케이션을 구동합니다.

$ mvn spring-boot:run
4

서비스 접속

BROWSER

브라우저를 열고 로컬 서버에 접속합니다.

Web UI (Main)
http://localhost:8080
H2 Console (DB)
http://localhost:8080/h2-console

초기 점검

VERIFY

시스템이 정상적으로 동작하는지 확인합니다.

  • "서버 추가" 버튼 클릭 및 정보 입력
  • "연결 테스트" 수행으로 SSH/Ping 확인
  • WebSocket 로그 스트리밍 정상 여부 확인
